## 谷歌 I/O 2026：Antigravity 2.0 正式入场

谷歌在年度开发者大会上正式发布 **Antigravity 2.0**，对标 Claude Code 和 Codex 的 AI Agent 编程平台。

**核心亮点：**

- **Agent-first 设计** — 以智能体为核心，专为 AI 编程任务构建
- **成本突破** — 从零构建完整操作系统的 Token 成本 **不足 $1000**
- **搜索深度整合** — AI Mode 接入编码能力，可直接生成可交互"迷你应用"
- **Gemini 3.5 Flash** — Token 生成速度约为其他模型的 **4倍**

**关键数据：**

| 指标 | 数值 |
|------|------|
| 月处理 Token 数 | 3.2 万亿（同比增长 7 倍） |
| Gemini 月活用户 | 9 亿 |
| Gemini Ultra 套餐 | $100/月 |

> 谷歌的入局标志着 AI 编程工具赛道从"双雄对峙"正式进入"三足鼎立"格局。Antigravity + Gemini 3.5 Flash 的组合有望对价格敏感的开发者和小型团队产生显著冲击。

## Anthropic 收购 Stainless

Anthropic 于 5 月 19 日宣布收购开发者工具公司 **Stainless**，估值至少 3 亿美元。

**Stainless 是什么：** 专注于将复杂 API 规范自动转化为多语言 SDK（TypeScript、Python、Go、Java、Kotlin）。**OpenAI、谷歌、Anthropic 三大头部厂商均在使用**。

**战略意图：**

- **"釜底抽薪"式卡位** — 直接控制竞争对手所依赖的底层工具链
- **强化 Claude 生态** — 帮助 Claude 更高效整合外部工具与数据
- **半年第四次收购** — 此前已收购 Bun、Vercept、Coefficient Bio

> 这是 AI 基础设施层的重大战略动作。Anthropic 不仅在模型层竞争，现在还向工具链基础设施延伸。SDK 生成赛道（Speakeasy、Liblab）有望迎来新机遇。

## xAI 正式入局：Grok Build 发布

马斯克 xAI 发布首款 Coding Agent **Grok Build**（早期 Beta 版）。

**核心功能：**

| 功能 | 说明 |
|------|------|
| **并行子智能体** | 大型任务自动分派，过程透明可追踪 |
| **工作流迁移** | 一键导入 Claude Code 配置（AGENTS.md、skills、MCP） |
| **计划模式** | 执行前征求用户意见，清晰 diff 展示更改 |
| **多媒体支持** | 终端内生成图像和视频 |

**定价：** $300/月（限时 $99/月，仅限 SuperGrok Heavy 用户）

> Grok Build 支持直接导入 Claude Code 配置，具有较强的生态迁移能力。加上马斯克此前拟以 600 亿美元收购 Cursor，xAI 在 AI 编程赛道的战略意图已非常清晰。

## OpenAI Codex 移动端扩展

OpenAI 将 Codex 编码工具引入 ChatGPT 移动应用，结合 **/goal 长时域模式**（持久化目标追踪、跨会话、预算控制），AI 编程从桌面端向移动端扩展。

> 移动端 AI 编程意味着"随时随地写代码"的能力边界进一步拓宽。这也是 OpenAI 扩大用户基数、抢占移动端市场份额的关键动作。
