## NVIDIA Q1 FY2027 财报：营收 $816 亿

5 月 20 日，NVIDIA 公布 FY2027 Q1 财报（截至 2026-04-26），营收达 **$816 亿**，同比增长约 85%，超分析师预期。数据中心业务指数级增长，Blackwell 架构出货顺畅。

### 关键信号

- **AI 交易远未耗尽** — 管理层明确表态，过去六个月内部人士零交易
- **Hopper → Blackwell 切换** — 供应瓶颈缓解，下游采购将加速
- **创纪录 $6250 亿净空头** — 短期博弈加剧，注意高估值压力

> NVIDIA 财报是 AI 赛道最核心的"水位线"。$816 亿验证全球 AI 基础设施投入仍在加速，对 AI Coding 工具的算力支撑和付费意愿有直接映射意义。

## OpenAI 推翻 80 年 Erdős 平面距离猜想

OpenAI 内部一个通用推理模型自主给出数学证明，推翻了 Paul Erdős 于 1946 年提出的**平面单位距离猜想**。证明核心使用代数数论构造，已由外部数学家独立核验通过。

### 为什么重要

- **真正的原创数学发现** — 不是"解题演示"，而是 AI 在严密逻辑推理上的质的跃升
- **外部独立核验** — 推理链可信度已达学术认可标准
- **对 AI Coding 的映射** — 数学推理是代码生成和算法优化的底层基础，预示 AI 编程在复杂算法设计、代码正确性证明方面的潜力

> 若推理模型可自主发现数学真理，则同样具备自主重构代码、发现 Bug 的能力，对 Claude Code、Copilot 等工具构成技术路线竞争压力。

## Claude Code Opus 4.7 发布：SWE-bench 87.6%

Anthropic 发布 Claude Code Opus 4.7，核心更新：

| 能力 | 更新内容 |
|------|----------|
| SWE-bench | 80.8% → **87.6%**（编程 Agent 新纪录） |
| 上下文窗口 | 上限扩至 **100 万 token** |
| 视觉分辨率 | 1.15MP → **3.75MP** |
| Task Budgets | 模型自主分配子任务 token 预算 |
| Background Agents | 隔离 Git worktree 并行执行 |

### 注意事项

**新 tokenizer 增 35% 输出 token**，月度费用可能隐性上升。

> 超过 87% 意味着 Claude Code 已能独立完成大多数编程任务，是生产可用的临界点信号。

## AI IDE 格局重塑：Cursor 3.1 vs Windsurf 2.0

| 维度 | Cursor 3.1 | Windsurf 2.0 |
|------|-----------|-------------|
| 核心能力 | 异步子 Agent + 多文件并行重构 | Agent Command Center 看板式管理 |
| 特色功能 | Bugbot 自动 Debug | Devin Cloud 一键部署 |
| 云端持久化 | — | 本地关机后云端持续运行 |
| 定价 | ~$15-20/月 | $20/月起 |

> AI IDE 竞争已从"代码补全质量"升级为"Agent 运行时架构"。深度重构选 Cursor；需要长时运行 + 云端部署选 Windsurf。

## OpenCode 突破 160K GitHub Stars

开源编程 Agent `anomalyco/opencode`（MIT 协议）突破 **160K Stars**，月活开发者达 750 万，贡献者超 900 人。

### 核心特性

- **75+ LLM 提供商**（含本地模型），真正模型无关
- **GitHub Copilot 订阅用户可免费接入**
- 事件溯源会话同步（SQLite）、TUI Mission Control
- 原生 MCP v1.4 RC 集成

> OpenCode 是首个达生产级规模的模型无关编程 Agent。Copilot 愿意向其提供免费接入，说明即使大厂也认可开源生态的重要性。
