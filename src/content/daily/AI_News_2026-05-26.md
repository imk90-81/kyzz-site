## 1. GPT-5.6 泄露：150 万 token 上下文 + UI 生成质变，6 月上线

**日期**：5 月 25-26 日持续发酵

**事件概要**：

多名开发者在 OpenAI Codex 后端日志中发现未官宣模型 GPT-5.6，内部代号 `iris-alpha`，另有 `ember-alpha`、`beacon-alpha` 两个变体。Polymarket 预测 6 月发布概率已超 85%。

| 参数 | GPT-5.5 | GPT-5.6（泄露） |
|------|---------|----------------|
| 上下文窗口 | 105 万 tokens | **150 万 tokens** ↑43% |
| 推理等级 | — | 最高支持 `x high` |
| UI 生成 | "AI味"明显 | **去 Slop 化**，像素级完美 |

**关键突破**：

- 实测输入 90 万 tokens 时模型仍正常响应，突破 1.05M 请求被完美处理
- UI 生成能力发生质变——在无详细 prompt 引导下自主生成了极简笔记应用「Lumen Notes」，像素级网格布局、克制色彩哲学、告别高饱和度"AI味"，被称为「UI de-slopification」
- OpenAI 主力版本迭代周期已压缩至 30-45 天，GPT-5.5 发布仅 3 周即泄露后继版本
- 双版本策略：标准版主打多步骤推理，Pro 版强化 Agent 工作流

> **为什么值得关注**：150 万 token 上下文意味着整个大型项目代码库可一次性送入模型，这是 AI 编码从"函数级辅助"跃迁至"项目级理解"的关键门槛。UI 生成质变更意味着 Vibe Coding 的输出质量正在逼近专业设计水准。

## 2. xAI Grok V9-Medium 训练完成：1.5T 参数 + Cursor 数据注入

**日期**：5 月 25 日

**事件概要**：

马斯克正式宣布 xAI 旗舰模型 Grok V9-Medium 训练完成，参数规模从 0.5T 直接跃升至 1.5 万亿，提升 3 倍，预计 2-3 周内（6 月中旬）发布。

| 训练阶段 | 状态 |
|---------|------|
| 预训练 | ✅ 已完成 |
| 补充训练（含 Cursor 代码数据） | ✅ 已完成 |
| 监督微调 SFT | 🔄 进行中 |
| 强化学习 RL | ⏳ 即将启动 |
| 公开发布 | 🎯 2-3 周内 |

**核心亮点**：

- 补充训练阶段引入 Cursor 代码数据——涵盖代码调用习惯、工程化思路、Bug 修复路径，代表了顶尖软件工程的真实实践
- 针对 NVIDIA Blackwell 架构深度优化，推理效率与成本大幅改善
- 定位"AI 工程师"角色——深度理解复杂代码库、自主完成复杂编程逻辑

> **为什么值得关注**：直接用 Cursor 真实编程数据训练是「换道超车」策略——不再依赖通用语料库，而是学习专业工程师在编程工具中的真实思维方式。这是首次有头部模型公开承认使用竞品编程工具数据训练。

## 3. DeepSeek Harness 团队正式亮相，国产 Coding Agent 进入产品化阶段

**日期**：5 月 19-21 日

**事件概要**：

DeepSeek 资深研究员陈德里在小红书发布招聘贴，证实内部正在组建全新的 Harness（代码智能体工程）团队，目标直指 Anthropic 旗下 Claude Code。

**团队定位**：

- 将前沿模型能力转化为领先的 Agent 产品
- 除模型本身以外的所有工作，都属于 Harness 的范畴
- 主攻桌面端智能体编程产品，内部核心对标 Claude Code

**背景**：

- DeepSeek V4-Pro API 永久降价 75%，输入仅 $0.435/百万 token
- 模型全部运行在华为昇腾芯片上，算力成本结构显著低于英伟达方案
- 近期「字符幻觉事件」已获官方回应，正在修复中

> **为什么值得关注**：DeepSeek 从"做模型"转向"做产品"，是国产 AI 编程赛道的关键信号。V4-Pro 的极致性价比 + Harness 团队的组建意味着中国开发者即将拥有一个本土化的 Claude Code 替代品。

## 4. Google Antigravity 2.0 上线：从 IDE 到"智能体优先工作平台"

**日期**：5 月 20 日（Google I/O 2026）

**事件概要**：

Google I/O 2026 发布 Antigravity 2.0 桌面应用 + CLI + SDK 全套工具链，核心定位从"面向开发者的智能体式 IDE"转向"通用智能体优先工作平台"。

**关键特性**：

| 特性 | 说明 |
|------|------|
| 多智能体协作 | 多个 AI 代理可并行协同工作 |
| 定时任务 | 支持设定周期性自动执行任务 |
| 自定义子智能体 | 用户可设计自定义工作流 Agent |
| 开放模型 | 不绑定单一模型，支持第三方 LLM |
| Antigravity CLI | 命令行工具，适合 CI/CD 集成 |
| 本地 SDK | 可本地部署的自定义工作流开发套件 |

**与 Jules 的配合**：

- Jules：异步自主编程智能体，开发者只需定义验收标准，Jules 在后台独立完成工程实现并提交 PR
- Antigravity 2.0：多智能体编排平台，开发者从"执行者"转变为"技术主管"
- 两者配合实现了开发范式的根本转变——从同步交互到异步编排

> **为什么值得关注**：Antigravity 2.0 开放模型的选择（不绑定 Gemini）直接对标 Claude Code 的封闭生态。多智能体编排 + Jules 异步执行 = "AI 后台工程团队"，这是目前所有 AI 编程工具中最完整的多智能体开发方案。
