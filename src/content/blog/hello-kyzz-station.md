你好，欢迎来到 Kyzz 空间站 🛸

这是我在技术宇宙中搭建的一座前哨站，用来记录探索与思考。

## 为什么建这个站

写博客这件事，我犹豫了很久。技术更新太快，文章容易过时；想法太零碎，又怕不够完整。但后来想通了——**写下来本身就是一种整理**，不完美的记录好过完美的遗忘。

这个站点的目标很简单：

- 📝 沉淀技术学习中的关键收获
- 🧩 记录项目实践中踩过的坑和解法
- 💡 分享那些"原来如此"的顿悟时刻

## 站点技术栈

本站基于以下技术构建：

- **Astro** — 静态站点生成，构建快、SEO 友好
- **Tailwind CSS** — 原子化样式，快速实现赛博风设计
- **MDX** — 在 Markdown 中使用组件，灵活扩展
- **Cloudflare Pages** — 全球 CDN 部署，访问飞快

## 接下来会写什么

计划覆盖的方向：

1. **前端工程化** — 构建工具、性能优化、架构设计
2. **编程语言** — TypeScript / Rust / Python 的深度使用心得
3. **系统设计** — 从零搭建可扩展系统的思考
4. **开源项目** — 参与和贡献开源的实践经验

## 最后

空间站已经上线，探索才刚刚开始。

如果你也是技术宇宙中的旅行者，希望这里的内容对你有所帮助。有什么想法，欢迎通过 [GitHub](https://github.com/kyzz) 或邮件联系我。
