🔥 头条
谷歌 I/O 2026 —— Antigravity 2.0 正式入场

谷歌在年度开发者大会 I/O 2026 上正式发布 Antigravity 2.0，这是其官方 AI Agent 编程平台，直接对标 Anthropic Claude Code 和 OpenAI Codex。谷歌的入局意味着 AI 编程赛道正式从"双雄对峙"进入"三足鼎立"格局。

  • 成本突破：从零构建完整操作系统的 Token 消耗成本不足 1000 美元
  • Gemini 3.5 Flash：Token 生成速度约为其他前沿模型的 4 倍，专为 Agent 和编码场景优化
  • 搜索深度整合：Google AI Mode 接入 Antigravity 编码能力，搜索界面可直接生成可交互"迷你应用"
值得关注的原因
Antigravity + Gemini 3.5 Flash 的成本效率组合对价格敏感的中小团队有显著吸引力，Claude Code 和 Codex 的市场份额或将受到直接冲击。

💰 战略级
Anthropic 3亿美元收购 Stainless：控制竞争对手的 SDK 工具链

Anthropic 于 5 月 19 日宣布收购开发者工具公司 Stainless（估值至少 3 亿美元，较 2024 年底的 1.5 亿美元翻倍）。Stainless 专注于将复杂 API 规范自动转化为 TypeScript、Python、Go、Java、Kotlin 等多语言 SDK——OpenAI、谷歌、Anthropic 三大厂商均在使用。

  • "釜底抽薪"式卡位：Anthropic 直接控制竞争对手依赖的底层工具链
  • 半年内第四次收购：此前已收购 Bun（JS 运行时）、Vercept、Coefficient Bio
  • Stainless 团队并入后，将帮助 Claude 更高效整合外部工具与数据
值得关注的原因
Anthropic 从模型层向工具链基础设施延伸的战略意图非常清晰。OpenAI 和谷歌短期内虽仍可继续使用 Stainless，但长期工具策略存在变数，SDK 生成赛道（Speakeasy、Liblab 等）有望迎来新机遇。

🚀 新品
xAI Grok Build Coding Agent 正式上线

马斯克 xAI 在内部动荡期间仍发布首款 Coding Agent Grok Build（早期 Beta 版），对标对象直指 Claude Code。

  • 无缝迁移：支持一键导入 Claude Code 的 AGENTS.md、skills、MCP servers 等配置，主打"无痛切换"
  • 并行子智能体：大型复杂任务自动分派给并行运行的子智能体，过程完全透明可追踪
  • 计划模式（Plan Mode）：执行前先征求用户意见，以清晰 diff 形式展示每项更改
  • 定价：$300/月（当前限时 $99/月，仅面向 SuperGrok Heavy 用户）
值得关注的原因
Grok Build 支持直接导入 Claude Code 配置，生态迁移成本极低。结合马斯克此前拟以 600 亿美元收购 Cursor，xAI 在编程赛道的战略意图已非常清晰。

📱 扩展
OpenAI Codex 移动端扩展 + /goal 长时域模式

OpenAI 此前将 Codex 编码工具引入 ChatGPT 移动端，结合 5 月初 Codex CLI 0.128.0 新增的 /goal 长时域模式（持久化目标追踪、跨会话、预算控制），AI 编程正在从桌面端向移动端延伸。

值得关注的原因
移动端 AI 编程能力对"随时随地写代码"的场景边界拓宽显著，尤其对移动学习和专业开发者有吸引力，有助于 OpenAI 扩大用户基数。
